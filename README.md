# NiduxOne

Sitio web oficial de NiduxOne, la plataforma todo-en-uno para comercio unificado.

## Tecnologias

- React + TypeScript
- Vite
- Tailwind CSS
- Material UI
- Radix UI

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy a GitHub Pages

```bash
npm run deploy
```

El sitio se publica en: https://zerocrc.github.io/niduxone/
